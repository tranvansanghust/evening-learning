# Evening Learning — Intelligent Learning System

An AI-powered learning application that helps users learn efficiently and track real progress through conversational quizzes, not just streaks.

**Key Features:**
- 📚 Track what you've truly learned (not just watched)
- 🤖 Adaptive oral-test quizzes via Telegram
- 📊 Personalized curriculum builder
- 💬 Chat-first interface (no need to open an app)

- **Database:** MySQL
- **Frontend:** React
